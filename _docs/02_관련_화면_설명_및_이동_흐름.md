## 01. 스플래시 - 앱 시작 화면 ⓑ

```
[전체 화면]
흰색 또는 연한 하늘색 배경

[중앙]
Oddo 로고
탄카츄 기본 정면 이미지

[하단]
“오늘의 마음을 함께 기록해요”
작은 로딩 점 3개
```

역할은 앱 실행 직후 브랜드를 보여주고 로그인 상태를 확인하는 화면입니다.

탄카츄는 캐릭터 시트의 **정면 또는 손 흔드는 모습**을 사용합니다.

이동 흐름:

```
앱 실행 → 로그인 상태 확인 → 로그인 또는 홈
```

## 02. 로그인 화면 ⓑ

```
[상단]
Oddo 로고
“오늘의 감정을 편하게 기록해보세요”

[중앙 카드]
이메일 입력
비밀번호 입력
로그인 버튼

[보조 영역]
비밀번호 찾기
회원가입

[소셜 로그인 영역]
카카오톡으로 계속하기
Google로 계속하기

[하단]
약관 및 개인정보 안내 문구
```

주요 더미 텍스트:

- 이메일 placeholder: `example@oddo.app`
- 비밀번호 placeholder: `비밀번호를 입력해주세요`
- CTA: `로그인`

역할은 기존 사용자가 앱 자체 계정 또는 소셜 계정으로 진입하는 화면입니다.

카카오 버튼은 실제 디자인 단계에서 노란색을 과하게 쓰기보다, 전체 톤을 깨지 않도록 **연한 배경 + 카카오 아이콘** 정도로 처리합니다. 메인 강조는 #3182F6 로그인 버튼에 둡니다.

이동 흐름:

```
로그인 성공 → 홈 화면
회원가입 선택 → 회원가입
비밀번호 찾기 → 비밀번호 찾기
로그인 실패 → 로그인 실패 모달
소셜 첫 로그인 → 추가 정보 입력
```

## 03. 회원가입 화면 ⓑ

```
[상단]
뒤로가기
제목: 회원가입
작은 안내: “Oddo와 함께 하루의 감정을 기록해요”

[본문 입력 영역]
이메일
비밀번호
비밀번호 확인
닉네임
생년월일
성별 선택
약관 동의 카드

[하단 고정]
회원가입 완료 버튼
```

필드 구성:

- 이메일: `hello@oddo.app`
- 비밀번호: `8자 이상 입력`
- 닉네임: `나의 별명`
- 생년월일: `2001.04.12`
- 성별: `여성 / 남성 / 선택 안 함`
- 약관:
  - 전체 동의
  - 서비스 이용약관
  - 개인정보 처리방침
  - 감정 분석 데이터 활용 동의

버튼:

```
회원가입 완료
```

역할은 앱 자체 계정을 생성하는 화면입니다.

이동 흐름:

```
로그인 → 회원가입 → 가입 완료 → 로그인 또는 홈
```

시작하기 버튼

역할은 카카오톡 또는 구글 첫 로그인 사용자의 필수 정보를 보완하는 화면입니다.

이동 흐름:

```
카카오/구글 첫 로그인 → 추가 정보 입력 → 홈
```

## 05. 로그인 실패 - 입력 오류 모달 ⓑ

```
[반투명 딤 배경]
로그인 화면 위에 모달

[중앙 모달 카드]
탄카츄 작은 미안한 표정
제목: “로그인 정보를 다시 확인해 주세요”
본문: “이메일 또는 비밀번호가 올바르지 않아요.”
버튼 1: 다시 입력하기
보조 텍스트 버튼: 비밀번호 찾기
```

공통 오류 톤은 부드럽게 유지합니다.

상황별 문구:

- 계정 없음: `가입된 계정을 찾을 수 없어요.`
- 비밀번호 오류: `비밀번호가 맞지 않아요. 다시 입력해볼까요?`
- 네트워크 오류: `잠시 연결이 불안정해요.`

이동 흐름:

```
로그인 실패 → 모달 → 다시 입력 또는 비밀번호 찾기
```

# 3. 홈 / 온보딩 진입 화면

## 07. 홈 화면 — 일기 미작성일 ⓐ

예시 이미지의 홈 화면 구성을 그대로 반영합니다.

```
[상단]
월 선택: 1월 2026 ▼
Oddo 로고
알림 아이콘
설정 아이콘

[주간 캘린더]
일 월 화 오늘 목 금 토
날짜 숫자
선택된 날짜는 원형 테두리
기록 여부는 작은 점으로 표시

[중앙]
탄카츄가 큰 카드 뒤에서 얼굴과 손을 내미는 구조

[메인 카드]
문서+연필 아이콘
“1월 14일”
“일기를 작성해보세요”
```

더미 데이터:

- 선택 날짜: `1월 14일`
- 상태: `아직 작성하지 않은 날`
- 카드 CTA 역할: 일기 작성 시작

이동 흐름:

```
로그인 → 홈
홈 미작성 카드 선택 → 일기 작성 시작 확인 모달
상단 알림 → 알림 목록
상단 설정 → 설정 홈
날짜 영역 → 월간 캘린더
```
