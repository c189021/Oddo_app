1단계 — 온보딩: "베이스라인을 잡는 법"부터 알려주기 (신규 사용자)
가입·로그인을 마치면(화면 1~6) 홈에 진입하지만(7), 아직 감정 측정 기준이 없는 상태라 Baseline 측정 필요 팝업이 뜹니다(8). 여기서 바로 측정에 들어가지 않고, 먼저 튜토리얼로 "어떻게 측정하는지"를 충분히 익히게 합니다.

- 권한 안내(9) 후 튜토리얼 1/5~5/5 진행(10~16)
- 통화 형식 연습 → 자기소개 기반 연습 → 음성 입력 연습 → 표정 입력 연습 → 측정 방식 안내·완료
- 즉, 실제 측정 전에 "말하기·표정·음성이 어떻게 분석되는지"를 미리 체험시켜 부담을 줄이는 단계
  2단계 — Baseline(기준값) 측정
  튜토리얼로 방식을 익힌 뒤 실제 기준값을 잡습니다(17~21).
- 얼굴·음성 Baseline 안내 → 준비 체크 → 영상통화 형식으로 측정 → 분석 중 → 완료
- 음성: pitch / 속도 / 에너지 측정
- 얼굴: FACS AU / 랜드마크 측정
- 이 값이 이후 일일 감정 분석의 비교 기준이 됩니다.
  3단계 — 심리 테스트
  개인 성향 데이터를 확보합니다(22~28).
- Big5 성격 검사 → MBTI 검사 → 성향 분석
- 중간 저장/이어하기 지원 → 완료
  4단계 — 챗봇 페르소나 선택
  함께 대화할 캐릭터(탄카츄)의 성격 톤을 고릅니다(29~31).
- 공감형 / 논리형 / 조언형 / 위트형 중 선택 → 설정 완료 → 온보딩 완료(32)

5단계 — 일일 감정 기록 (기존 사용자, 매일 반복되는 핵심 루프)
온보딩을 마친 사용자는 매일 캘린더 기반 홈에서 날짜를 선택해(33~35) 다음 4스텝을 진행합니다.
Step 1. 말하기 — 캐릭터(탄카츄)와 영상통화 형식으로 실시간 대화하며 하루를 이야기 → STT로 텍스트 변환 → AI가 요약 생성
Step 2. 확인하기 — AI 요약 확인 후 사용자가 직접 검토·수정 → 내용이 부족하면 챗봇이 추가 질문을 던져 일기를 보완
Step 3. 영상 제작 — 완성된 일기를 토대로 3인칭 시나리오 생성 → Text-to-Video → 숏폼 영상 제작 → 사용자가 자신의 하루를 담은 영상을 시청
Step 4. 상담하기 — 영상 시청 후 다시 캐릭터와 영상통화 형식으로 상담 진행(SFT 기반 상담챗봇) → 행동 가이드 제공 → 감정 리포트 생성으로 마무리
이후 홈은 "일기 작성일" 상태로 갱신되고(48), 일기 상세·감정 리포트·상담 기록을 다시 열람할 수 있습니다.
